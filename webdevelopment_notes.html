HTML Basics

HTML defines the structure of a webpage. Uses tags like <div>, <p>, <img>.

CSS Styling

Controls the layout, colors, fonts, spacing, animations.

body{
  background:#f5f5f5;
  font-family:Arial;
}
.card{
  box-shadow:0 4px 10px rgba(0,0,0,0.1);
}

Responsive Design

Ensures pages look good on mobile, tablet, and desktop.

@media(max-width:768px){
  .container{flex-direction:column;}
}
