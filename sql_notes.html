SQL Introduction

SQL (Structured Query Language) is used to create, read, update, and delete data in databases.

SELECT * FROM employees;

DDL — Data Definition Language

CREATE TABLE student(
  id INT PRIMARY KEY,
  name VARCHAR(50),
  marks INT
);

ALTER TABLE student ADD email VARCHAR(80);
DROP TABLE student;

DML — Data Manipulation Language

INSERT INTO student VALUES(1,'Naveen',90);
UPDATE student SET marks = 95 WHERE id = 1;
DELETE FROM student WHERE id = 1;

Joins

Joins combine rows from two or more tables.

SELECT e.name, d.dept
FROM emp e
JOIN dept d ON e.dept_id = d.id;

Subqueries

SELECT name
FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);

Window Functions

SELECT name, salary,
RANK() OVER(ORDER BY salary DESC) AS rnk
FROM employees;

Constraints

  • PRIMARY KEY
  • FOREIGN KEY
  • UNIQUE
  • NOT NULL
  • CHECK

Views

CREATE VIEW high_salary AS
SELECT * FROM employees WHERE salary > 50000;

Indexing

Improves search performance.

CREATE INDEX idx_name ON employees(name);

  • Consistency

  • Durability
