Introduction & Setup

Install Python (3.10+ recommended). Use venv for virtual environments.

python -m venv env
# activate (Windows)
env\Scripts\activate
# activate (mac/linux)
source env/bin/activate
pip install -r requirements.txt

Basic Syntax

Printing, variables, comments.

print('Hello world')
# variables
x = 10
name = 'Naveen'

Data Types

Numbers, strings, lists, tuples, sets, dicts — examples:

# list
fruits = ['apple','banana','mango']
# dict
student = {'name':'Naveen','age':25}

Control Flow

If/else, loops, comprehensions.

for i in range(5):
    print(i)

# list comprehension
squares = [x*x for x in range(10)]

Functions

Define and call functions, default args, *args, **kwargs.

def greet(name='Guest'):
    return f'Hello {name}'

print(greet('Naveen'))

Object-Oriented Programming

Classes, inheritance, magic methods.

class Student:
    def __init__(self,name,age):
        self.name=name
        self.age=age

    def greet(self):
        return f'Hi {self.name}'
